# Introduction

Dans le cadre d'un **projet en Instrumentation Avancée** au département Mesures Physiques de Marseille, nous avons réalisé en **quadrinôme** un prototype de **capteur connecté de mesure de puissance** dont l'objectif était de **mesurer** une masse dans un intervalle compris entre 0 et 1000g et de **transmettre** les données à une **interface Node-RED** via un **protocole MQTT**.
